üéØ How to Play Guess the Photo

A visual guessing game where everyone reveals tiles to identify the hidden photo!

  • Setup: Host uploads one or more photos (events, places, products, anything fun)
  • Configure Grid: Choose number of rows and columns (max 30x30)
  • Create Board: Grid tiles automatically cover the photo
  • Play: Players take turns clicking tiles to reveal parts of the photo
  • Guess: After each reveal, try to identify what the photo is
  • Win: Successfully identify the photo before all tiles are revealed!
  • Reset: Use the Reset Board button to cover all tiles again
  • Multiple Rounds: Upload multiple images and use Next Image button to move to the next feature

üí° Tips:

  • Start with fewer tiles (5x8) for easier games
  • Use more tiles (20x30) for challenging rounds
  • Great for sprint reviews, feature onboarding, and team building!
